# Peer Bring

Student-exclusive social marketplace with AI-powered negotiation.

## Features

- Home dashboard with study reminders, active negotiations, and friend activity
- Community social feed with groups
- Marketplace for buying/selling student items
- Hagglit AI — automated negotiation bot
- Real-time-style messaging with deal chat
- User profile with reputation and deal history

## Run Locally

Just open `index.html` in any browser — no build step needed.

## Tech Stack

HTML5 · CSS3 (custom properties) · Vanilla JS · Lucide Icons · Google Fonts
